jquery.checkboxall v1.0

Description

The jquery.checkboxall plugin turns all input checkboxes in the container. You can use the first checkbox at the same time enable/disable additional checkboxes for the container.

The first checkbox class name is by default: all. This is required, otherwise the plugin will not work and your browser will work console will indicate an error.

You can also use a different class name, but you must pass it to the checkboxall function.

Method

$(selector).checkboxall(optional-class-name);
